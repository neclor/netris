# Netris

[netris.neclor.com](https://netris.neclor.com/) · [GitHub Pages](https://neclor.github.io/netris)

## About

A Tetris-like game built with Godot.  
Netris is a block-stacking puzzle game with its own twists on the classic formula. Stack falling pieces, clear lines, and compete for the top score on the global leaderboard.
